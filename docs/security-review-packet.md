# Security Review Packet

## Security Questionnaire Response Pack

Reusable procurement-ready security questionnaire responses are available at:

- `docs/readiness-binder/security-questionnaire-responses.md`

Use this response pack for hospital vendor reviews, enterprise cybersecurity assessments, and procurement questionnaires. It references implemented controls, evidence PRs, current status, residual gaps, and planned improvements.

Reviewer note: this packet uses readiness and evidence language only. It does not claim SOC 2 certification, HITRUST certification, HIPAA certification, or any other formal certification.
